## Why Packen?

Packen is designed to be the ultimate bundling solution for modern JavaScript and TypeScript projects. It combines the power of Rollup with cutting-edge transformers to deliver exceptional performance and developer experience.

## Quick Start

Get started with Packen in seconds:

```bash
# Install Packen
npm install --save-dev @visulima/packem

# Initialize configuration
npx packem init

# Build your project
npx packem build
```

## Key Features

### üöÄ **Performance First**
- Built on Rollup with optimized plugins
- Support for multiple high-performance transformers
- Parallel processing and caching
- Bundle analysis and visualization

### üì¶ **Package.json Exports**
- Automatic entry point detection from `package.json#exports`
- Support for `main`, `module`, and `types` fields
- Multiple export conditions (import/require, development/production)
- Automatic Node.js 10 compatibility generation

### üîÑ **Multiple Transformers**
- **esbuild** - Lightning fast JavaScript/TypeScript transformer
- **swc** - Rust-based super-fast compiler
- **OXC** - Next-generation JavaScript toolchain
- **sucrase** - Fast alternative to Babel
- **TypeScript** - Official TypeScript compiler

### üé® **CSS Processing**
- **Sass** (.scss/.sass files)
- **Less** (.less files)  
- **Stylus** (.styl files)
- **PostCSS** with plugin ecosystem
- **CSS Modules** with scoped styling
- **Lightning CSS** for optimization
- **cssnano** for minification

### ‚öõÔ∏è **React Integration**
- Server Components with `"use server"`
- Client Components with `"use client"`
- Automatic chunk splitting for boundaries
- Shared modules across runtimes

### üîß **Developer Experience**
- Watch mode for development
- Source maps support
- TypeDoc documentation generation
- ESM ‚áÑ CJS interoperability
- Environment variable injection

## Packages

Packen is built as a collection of focused packages:

| Package | Description |
|---------|-------------|
| [`@visulima/packem`](/docs/core/) | Main bundler package with CLI |
| [`@visulima/packem-rollup`](/docs/rollup/) | Rollup plugins and transformers |
| [`@visulima/rollup-plugin-css`](/docs/css/) | CSS processing and optimization |
| [`@visulima/css-style-inject`](/docs/css/style-inject) | Runtime CSS injection utility |
| [`@visulima/packem-share`](/docs/utils/) | Shared types and utilities |
