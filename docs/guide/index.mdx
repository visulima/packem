## What is Packen?

Packen is a fast and modern bundler for Node.js and TypeScript projects. Built on top of Rollup, it combines the power of proven bundling technology with cutting-edge transformers like esbuild, swc, OXC, and sucrase.

## Core Concepts

### Package.json Driven

Packen uses your `package.json` as the source of truth for builds:

```json
{
  "type": "module",
  "main": "./dist/index.cjs",
  "module": "./dist/index.js",
  "types": "./dist/index.d.ts",
  "exports": {
    ".": {
      "types": "./dist/index.d.ts",
      "import": "./dist/index.js", 
      "require": "./dist/index.cjs"
    }
  }
}
```

### Transformer Architecture

Choose from multiple transformers based on your needs:

- **esbuild** - Fastest compilation, great for most projects
- **swc** - Rust-powered with excellent performance
- **OXC** - Next-generation toolchain (experimental)
- **TypeScript** - Official compiler with full type checking
- **sucrase** - Lightweight and simple

### Modern JavaScript Support

- ES2015+ syntax transformation
- JSX and TSX support
- Dynamic imports and code splitting
- Tree shaking for optimal bundle size

## Use Cases

### Library Development

Perfect for creating npm packages:

```typescript
// Automatic dual package support
export function myLibrary() {
  return 'Hello from my library!'
}
```

### Component Libraries

Build React, Vue, or framework-agnostic components:

```typescript
// React component with CSS modules
import styles from './Button.module.css'

export function Button({ children, ...props }) {
  return <button className={styles.button} {...props}>{children}</button>
}
```

### CLI Tools

Create command-line tools with proper bundling:

```typescript
#!/usr/bin/env node
import { program } from 'commander'

program
  .command('build')
  .action(() => {
    console.log('Building...')
  })
```

### Node.js Modules

Build server-side libraries and utilities:

```typescript
// Server utilities with proper externals
export { createServer } from './server'
export { database } from './db'
```

## Why Choose Packen?

### Performance First

- **Fast Transforms**: Multiple high-performance transformers
- **Incremental Builds**: Only rebuild what's changed
- **Parallel Processing**: Leverage all CPU cores

### Developer Experience

- **Zero Configuration**: Works immediately with sensible defaults
- **Rich Diagnostics**: Clear error messages and warnings
- **Watch Mode**: Fast rebuilds during development

### Modern Tooling

- **TypeScript Native**: First-class TypeScript support
- **CSS Pipeline**: Complete styling solution
- **Tree Shaking**: Automatic dead code elimination

### Ecosystem Integration

- **Rollup Compatible**: Use any Rollup plugin
- **Framework Agnostic**: Works with React, Vue, Svelte, and more
- **Monorepo Friendly**: Perfect for multi-package workspaces
