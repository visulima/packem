## What is Packen?

Packen is a fast and modern bundler for Node.js and TypeScript projects. Built on top of Rollup, it combines the power of proven bundling technology with cutting-edge transformers like esbuild, swc, OXC, and sucrase.

### Key Concepts

- **Zero Configuration**: Works out of the box with sensible defaults
- **Multiple Transformers**: Choose from esbuild, swc, OXC, sucrase, or TypeScript
- **Package.json Driven**: Uses your package.json exports to determine build targets
- **Multi-Runtime**: Supports Node.js, browser, edge-light, and react-server environments
- **CSS First-Class**: Built-in support for CSS, Sass, Less, Stylus, and CSS Modules
