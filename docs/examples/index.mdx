## Quick Start Templates

Ready-to-use templates for common project types:

### TypeScript Library
```bash
# Clone the template
git clone https://github.com/visulima/packem-template-typescript-library
cd packem-template-typescript-library
npm install
npm run build
```

## Contributing Examples

Have a great example to share? We'd love to include it!

1. **Fork the repository**
2. **Create your example** in the appropriate category
3. **Include complete source code** and configuration
4. **Add clear documentation** explaining the example
5. **Submit a pull request**

### Example Structure

Each example should include:

- `README.md` - Overview and instructions
- `package.json` - Dependencies and scripts
- `packem.config.ts` - Packen configuration
- `src/` - Source code
- `dist/` - Expected output (optional)

### Documentation Requirements

- **Clear title and description**
- **Prerequisites and setup instructions**
- **Step-by-step explanation**
- **Expected output and results**
- **Troubleshooting tips**
- **Related examples and links**
