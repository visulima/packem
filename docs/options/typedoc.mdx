# typedoc

Options for generating API documentation with [TypeDoc](https://typedoc.org/). Set to `false` to disable.

**Type:** `TypeDocumentOptions | false`  
**Default:** `{}`

## Quick Setup

Use the `add` command to automatically configure TypeDoc:

```bash
npx packem add typedoc
```

This will:
- Add TypeDoc builder configuration to your `packem.config.ts`
- Install TypeDoc and related plugins
- Set up the builder for documentation generation

## Example

```typescript
export default defineConfig({
  typedoc: {
    format: 'markdown',
    output: 'docs/api'
  }
})
```
