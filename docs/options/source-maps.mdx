# Source Maps

Source maps provide a way to map compiled code back to the original source code, enabling debugging of the original code in browsers and other tools. Packen supports various source map formats and configurations.

## Basic Configuration

### Enable Source Maps

```typescript
export default defineConfig({
  rollup: {
    output: {
      sourcemap: true
    }
  }
})
```

### Source Map Types

```typescript
export default defineConfig({
  rollup: {
    output: [
      {
        format: 'esm',
        file: 'dist/index.mjs',
        sourcemap: true // External source map
      },
      {
        format: 'cjs',
        file: 'dist/index.cjs',
        sourcemap: 'inline' // Inline source map
      },
      {
        format: 'iife',
        file: 'dist/index.global.js',
        sourcemap: 'hidden' // No reference comment
      }
    ]
  }
})
```

## Advanced Configuration

### Source Map Paths

```typescript
export default defineConfig({
  rollup: {
    output: {
      sourcemap: true,
      sourcemapPathTransform: (relativeSourcePath, sourcemapPath) => {
        // Transform source paths in source maps
        return relativeSourcePath.replace('../src/', 'src/')
      }
    }
  }
})
```

### Source Content Inclusion

```typescript
export default defineConfig({
  rollup: {
    output: {
      sourcemap: true,
      sourcemapExcludeSources: false, // Include source content
      sourcemapFile: 'dist/index.js.map'
    }
  }
})
```

### Base URL Configuration

```typescript
export default defineConfig({
  rollup: {
    output: {
      sourcemap: true,
      sourcemapBaseUrl: 'https://cdn.example.com/src/'
    }
  }
})
```

## Source Map Validation

### Verification

```typescript
export default defineConfig({
  hooks: {
    'build:after': async () => {
      // Validate source maps
      const { SourceMapConsumer } = await import('source-map')
      const fs = await import('fs/promises')
      
      const mapContent = await fs.readFile('dist/index.js.map', 'utf-8')
      const consumer = await new SourceMapConsumer(JSON.parse(mapContent))
      
      // Test source map accuracy
      const originalPosition = consumer.originalPositionFor({
        line: 1,
        column: 0
      })
      
      console.log('Source map validation:', originalPosition)
      consumer.destroy()
    }
  }
})
```

### Testing Tools

```typescript
export default defineConfig({
  rollup: {
    plugins: [
      {
        name: 'sourcemap-test',
        generateBundle(options, bundle) {
          // Test source map generation
          Object.keys(bundle).forEach(fileName => {
            if (fileName.endsWith('.map')) {
              const map = JSON.parse(bundle[fileName].source)
              console.log(`Source map for ${fileName}:`, {
                version: map.version,
                sources: map.sources.length,
                mappings: map.mappings.length
              })
            }
          })
        }
      }
    ]
  }
})
```

## Troubleshooting

### Missing Source Maps

<Callout type="warning">
If source maps aren't working, ensure all transformers have source map support enabled.
</Callout>

```typescript
export default defineConfig({
  // Enable source maps at all levels
  rollup: {
    output: {
      sourcemap: true
    }
  },
  
  esbuild: {
    sourcemap: true
  },
  
  css: {
    sourceMap: true
  }
})
```

### Incorrect Source Paths

```typescript
export default defineConfig({
  rollup: {
    output: {
      sourcemap: true,
      sourcemapPathTransform: (relativeSourcePath) => {
        // Fix source paths
        return relativeSourcePath.replace(/\\/g, '/')
      }
    }
  }
})
```

### Large Source Maps

```typescript
export default defineConfig({
  rollup: {
    output: {
      sourcemap: true,
      // Exclude node_modules from source maps
      sourcemapIgnoreList: (relativeSourcePath) => {
        return relativeSourcePath.includes('node_modules')
      }
    }
  }
})
```
